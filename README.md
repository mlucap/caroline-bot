## Function

Passes the user's message through a function that changes it and makes it 'uwu'. 

Input: `>I love this!` 

Result: `I wuv this ʕ￫ᴥ￩ʔ`

## How to run 

`npm i` or `yarn`

Create a file called `.env` and it should look like this:

```
TOKEN=token
ID=your id
PREFIX=desired prefix
```

Run `node index.js`
